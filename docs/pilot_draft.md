# 长期情感陪伴能力评测 Pilot 草案

## 1. Pilot 想回答的问题

1. 在多次重复互动中，哪个模型更能提供稳定的情绪支持？
2. 当同一问题反复出现时，哪个模型更能保持耐心和连续性？
3. 哪个模型对历史信息的利用更自然、更像长期关系中的回应？
4. 当用户出现失望、冷淡或轻度质疑时，哪个模型更有修复能力？

## 2. Session 结构

- Session 1：建立联系
- Session 2：关系延续与重复情绪
- Session 3：挑战、失望、疏离或修复

## 3. 重点维度

- 共情准确性
- 回应自然度
- 需求匹配
- 非模板感
- 跨 session 连续性
- 历史信息调用质量
- 语气与人格稳定性
- 关系修复能力
- 边界感
- 用户是否愿意继续聊下去
